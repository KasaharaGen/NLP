Q: A Windows' Ctrl+Z equivalent for OSX On many operating systems if you want to undo something you can press Ctrl+Z. I tried this on Mac OS X and it does not work. What is the suitable equivalent?

A: On a Mac, many Windows CTRL + xxx shortcuts are changed to CMD + xxx.
If you are a new switcher from Windows to Mac, check out the following switcher guide. Full of useful tips!
http://lifehacker.com/224674/hack-attack-a-guide-for-switching-to-a-mac

A: Command + Z check Edit menu if you are unsure if the shortcut is available for a program or a scenario. 
On a side note, Command works as the default shortcut key on Mac OS X where Windows' default modifier key is Ctrl
This holds for Copy, Paste, Open, etc...
