Q: Is there a way to use my USB web Camera which I use with my windows PC to use with my iPad first edition? Is there a way to use my USB web Camera which I use with my windows PC to use with my iPad first edition?
Or any other camera to use with ipad first edition.

A: The iPad does not support using external cameras with it, as it does not have the drivers required.
Thus, the iPad cannot use an external camera. However, the new iPad 2 has cameras.
