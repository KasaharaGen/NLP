Q: How can I see my IPv6 neighbors? On a Linux machine, I can see my IPv6 neighbors using the following command (Also see the Linux IPv6 HOWTO:
    # ip -6 neighbor show
    2002:333:333::1 dev eth1 lladdr 00:12:1e:33:AA:BB router REACHABLE

What is the equivalent command on a Mac (currently running Snow Leopard)?
